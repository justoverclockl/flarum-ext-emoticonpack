### Installation

Install with composer:

```sh
composer require justoverclock/flarum-ext-emoticonpack:"*"
```

### Updating

```sh
composer update justoverclock/flarum-ext-emoticonpack:"*"
php flarum migrate
php flarum cache:clear
```
